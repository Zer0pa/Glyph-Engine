# Public Audit Limits

## What This Scaffold Can Establish

| Surface | What It Can Show |
|---|---|
| Docs | current hold/extraction posture |
| Source boundary | candidate source families and exclusions |
| GPD state | next autonomous work gate |

## What It Cannot Establish

| Missing input | Why it matters |
|---|---|
| extracted package code | no install or API claim can close |
| benchmark/ablation output | no superiority claim can close |
| license authority | no public promotion can close |
| rights-cleared corpus assets | no image-bearing release can close |

Do not infer library quality, commercial value, or cross-script generality from
this scaffold alone.
