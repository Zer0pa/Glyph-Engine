# Governance

## Scope

This scaffold governs hold/extraction decisions for `gnosis-glyph-engine`.

## Truth Hierarchy

1. `SOVEREIGN_PRD.md`
2. `SOURCE_BOUNDARY.md`
3. repo-local artifacts created by future phases
4. README and supporting docs

## Decision Rights

| Topic | Authority | Notes |
|---|---|---|
| Extraction boundary | repo owner or delegated maintainer | must update PRD and source boundary |
| Public package claims | blocked | requires ablation and smoke proof |
| Release approval | blocked | requires license and promotion gates |
| Legal statements | owner | final license is owner-deferred |

## Status Vocabulary

Use `UNTESTED`, `PARTIAL`, `BLOCKED`, `OWNER_DEFERRED`, and `PASS` only when an
artifact path supports the state.
