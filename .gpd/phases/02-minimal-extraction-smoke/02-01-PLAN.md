# Phase 02 Plan 01: Borrowed-Baseline Sanity Path

## Objective

Prove the frozen Phase 01 interface is implementable and monorepo-free by
running the three borrowed-baseline arms end-to-end, from a repo-local
synthetic fixture through glyph-engine descriptors into morph-bench's
evaluator, producing a JSON ablation report.

## Tasks

1. Add `pyproject.toml` declaring `gnosis_glyph_engine` with src-layout and
   borrowed-dep pins.
2. Implement `src/gnosis_glyph_engine/protocols.py` exactly matching the
   Phase 01 D-02 signature.
3. Implement `src/gnosis_glyph_engine/fixtures.py::synthesize_twelve_glyphs`
   — deterministic, 3 families × 4 variants, 64×64 uint8.
4. Implement `src/gnosis_glyph_engine/baselines/{orb,hu_regionprops,hog}.py`
   using only OpenCV / scikit-image / scikit-learn.
5. Implement `src/gnosis_glyph_engine/manifest_builder.py::build_manifest`
   emitting morph-bench schema v1 dicts.
6. Add `src/gnosis_glyph_engine/owned/__init__.py` as a placeholder that
   raises `SourceRetrievalPending` until D-06 clears.
7. Add `tests/` covering fixture determinism, each baseline's contract,
   manifest shape.
8. Add `scripts/run_ablation.py` that:
   - generates the fixture,
   - builds one manifest per baseline arm,
   - invokes morph-bench CLI per manifest,
   - aggregates per-arm metrics into
     `artifacts/ablation/ablation_report.json`.
9. Provision RunPod workspace. Rsync this scaffold and the sibling
   morph-bench scaffold. Install both in a Python 3.13 venv.
10. Run `pytest` then `scripts/run_ablation.py` on the pod.
11. Record the path-rewrite ledger (none for borrowed code; that column is
    empty until D-06 Phase 02b).
12. Commit code to GitHub; upload artefacts to HF dataset.

## Pass Condition

Phase 02a passes when pod-side `pytest` + `run_ablation.py` both exit
successfully, and `ablation_report.json` contains metric rows for
`baseline_orb`, `baseline_hu_regionprops`, `baseline_hog`, each with finite
`sigma`, `null_mean`, `null_std`, and a `mean_jaccard` from leave-25%-out.

## Non-Goals For This Plan

- No `package_boundary_earned` verdict. That requires the owned arms from
  Phase 02b.
- No performance claim. Three runs on 12 synthetic items do not generalise.
- No public promotion. Authority status unchanged.
