# Notation Glossary

| Term | Meaning |
|---|---|
| `package_boundary_earned` | Binary authority metric for whether this lane deserves an extracted package. |
| admitted consumer | A workstream or application that demonstrably needs the extracted interface. |
| descriptor ablation | A comparison showing whether owned descriptors add value over borrowed baselines. |
| smoke fixture | Tiny synthetic or rights-clear input used to prove monorepo-free execution. |
